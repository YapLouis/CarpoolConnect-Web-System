This repository showcases full-stack web applications developed with a focus on relational database integrity, server-side processing, and dynamic user interfaces. The primary project, APU Carpool Connect, is a web-based platform designed to solve campus commuting challenges through data-driven solutions.

Core Technical Features:
🗄️ Relational Database Design: Implementation of complex SQL schemas, ensuring data normalization and transactional integrity.

📍 Geospatial Integration: Use of Leaflet.js for real-time map visualization and coordinate-based location tracking.

⚙️ Server-Side Logic: Developed using PHP and JavaScript, implementing the Post-Redirect-Get (PRG) pattern to prevent form resubmission errors.

💰 Automated Calculations: Integrated logic for automated fare calculations based on distance and route parameters.

📱 Responsive UI/UX: Built with a focus on clean, accessible interfaces for both desktop and mobile users.

Tech Stack:
Languages: PHP, JavaScript, SQL

Frontend: HTML5, CSS3, Leaflet.js

Database: MySQL / MariaDB

Architecture: Client-Server Model
